Grocery Services

"SaveCart"


A service that scans local grocery stores for items on sale, compares it to a recipe database, and presents weekly menus using those ingredients.

This project is inspired from an earlier project, which scanned, formatted, and sent out the lunch/dinner menus at UCR's residential restaurants.


Python

Heroku

Raspberry Pi

Objectives

- Provides users with a weekly recipe list of ~15 recipes

- Uses geolocation to determine grocery stores to scan

- Able to scrape common grocery chain websites for on-sale items

- Allows shopping from only one store

- Allows users to set "tastes", meal price ranges, and stores scanned

- Fast and simple signup and unsubscription

- Webpage, Email, and SMS interfaces
