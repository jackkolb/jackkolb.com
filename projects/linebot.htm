LINE ChatBot

"Mademoiselle L'Bot"


A LINE chatbot with an extensive array of features, responses, and functions.


Python

LINE API

Heroku

Google Drive APIs

Objectives

- Runs through the LINE API, interfaces through the LINE app

- Connects to a Google Drive database for variable storing and recalling

- Appropiately formats recalled data, as per the task

- Can set timers and alarms for custom or variable times, recalculates times to EST

- Timers and alarms are automatically stored and recalled when the server resets

- Permission levels set for various groups

- Utilizes a variety of message types

- Capable of pre-filling and submitting google forms
